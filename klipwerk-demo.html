Why Klipwerk
Everything you need.
Nothing you don't.
⚡
Raw ffmpeg Speed
No re-encoding unless you want it. Stream-copy trimming is instant. CRF + preset control when quality matters.
✂
Live Crop Preview
Drag a crop rectangle directly on the preview. Rule-of-thirds overlay, aspect-ratio presets (16:9, 9:16, 1:1, 4:3), and pixel-precise inputs.
📊
Waveform Scrubber
Vectorized peak extraction gives you a real audio waveform on the timeline with hover thumbnails and In/Out markers.
🎞
Sequence Export
Add multiple In/Out clips to the Klip list, reorder via drag-and-drop, and concatenate them into one output in a single click.
🎛
7 Output Formats
H.264, H.265, AV1, VP9 across MP4, MKV, and WebM. Prefix/suffix output naming. Full ffprobe info panel per file.
⌨
Keyboard-First
Every action has a shortcut. I/O to mark, C to clip, Space to play, arrow keys to scrub. Your hands never leave the keyboard.
Get Started
Install in under a minute
1
Install Python if you don't have it yet
Download and install Python 3.10 or newer from python.org/downloads.
On Windows: ✅ check "Add Python to PATH" during installation!
→ verify: open a terminal and type python --version
2
Install ffmpeg the engine under the hood
Download ffmpeg-release-essentials.zip from gyan.dev, extract it, and place ffmpeg.exe + ffprobe.exe somewhere on your PATH, or just drop them next to the Klipwerk folder.
If you have Homebrew: run brew install ffmpeg in Terminal. No Homebrew? Get it at brew.sh first.
Run sudo apt install ffmpeg (Ubuntu/Debian) or the equivalent for your distro. Done.
3
Download Klipwerk
Click the green Code → Download ZIP button on GitHub, extract the ZIP somewhere you like.
4
Install dependencies one command, once
Open a terminal inside the extracted folder and run:
# installs PyQt6, OpenCV, NumPy, Pillow automatically
pip install -e .
All dependencies install automatically, no manual pip installs needed. You only need to do this once.
5
Launch Klipwerk
Just double-click run.pyw in the Klipwerk folder, no terminal needed, no black console window. 🎉

Alternatively: klipwerk in any terminal works too after the install.
Run klipwerk in Terminal from any directory, or python -m klipwerk inside the folder.
Run klipwerk in your terminal from any directory, or python -m klipwerk inside the folder.
